Neural Networks

What is it?

A way for computers to learn from examples, like a brain made of math.

How it works

INPUT Receives data (numbers, pixels, words)

HIDDEN Processes and finds patterns

HIDDEN Deep layers = deep learning

OUTPUT Gives the answer

Key concepts

Weights — connections between nodes, strengthened when useful

Training — adjust weights until answers are correct

Layers — more layers = more complex patterns

Uses

Image recognition, language, recommendations, predictions
