The Explanation

What is HexClamp?

An autonomous loop that never stops. It keeps running, learning, and improving with each cycle.

How It Works

OBSERVE — Look around. What just happened? What's the current state?

CONDENSE — Filter out the noise. Keep only what's relevant.

PLAN — Decide: what should we do next?

EXECUTE — Do it. Run the plan.

VERIFY — Did it work? Find any problems?

PERSIST — Save the results. Remember for next loop.

The Loop

After PERSIST, it loops back to OBSERVE and starts again. Forever.

If VERIFY finds a problem, it loops back to PLAN to try again.
