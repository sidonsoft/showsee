HexClamp

What is it?

An autonomous loop that keeps running, learning, and improving. Forever.

How it works

OBSERVE Look around. What's happening?

CONDENSE Filter noise. Keep what's relevant.

PLAN Decide what to do next.

EXECUTE Run the plan.

VERIFY Did it work? Find problems?

PERSIST Save results for next loop.

The loops

PERSIST → back to OBSERVE (main loop).

VERIFY fails → back to PLAN (retry loop).
