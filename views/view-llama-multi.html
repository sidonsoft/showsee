What is LLaMA?

Meta's open source language model. "Large Language Model Meta AI." Weights are free to download and run locally.

Anyone can run a GPT-class AI on their own hardware. No API costs. No data leaving your machine.

LLaMA 4 Models (April 2025)

Llama 4 Scout

17B activeMoEmultimodal

Fits in single H100 GPU. 10M token context. Best in class for its size.

Llama 4 Maverick

17B activeMoEmultimodal

Most capable for general use. Matches GPT-4o on benchmarks. Cost-effective.

Llama 4 Behemoth

trainingMoE

Frontier model still in training. Expected to rival GPT-4.5 class.

Key Points

  • MoE Architecture — only activates needed experts, faster inference
  • Native Multimodal — sees images and text together (early fusion)
  • Open Weights — download, modify, run locally, fine-tune
  • Massive Context — up to 10M tokens (Scout)
